# テクスチャ画像について

このフォルダには、こけしのパーツ用のテクスチャ画像を配置します。

## フォルダ構成

- `faces/` - 顔用テクスチャ
- `bodies/` - 胴体用テクスチャ
- `patterns/` - 共通パターン

## 画像仕様

- **推奨サイズ**: 512x512px以上
- **対応形式**: JPG, PNG, GIF
- **ファイル名**: 英数字とハイフン、アンダースコアのみ

## 必要なファイル

### faces/フォルダ
- basic.jpg / basic_preview.jpg
- smile.jpg / smile_preview.jpg  
- traditional.jpg / traditional_preview.jpg

### bodies/フォルダ
- floral.jpg / floral_preview.jpg
- geometric.jpg / geometric_preview.jpg
- traditional.jpg / traditional_preview.jpg

プレビュー画像は128x128pxの小さなサムネイル版を推奨します。
